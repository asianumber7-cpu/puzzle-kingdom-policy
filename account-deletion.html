1. 삭제 요청 대상

퍼즐 킹덤 앱을 이용하면서 생성된 계정 정보, 서버 동기화 또는 계정 기능이 활성화된 경우의 서버 측 게임 진행 데이터, 구매 처리 기록, 문의 이메일 기록 등의 삭제를 요청할 수 있습니다.

기기 내부에만 저장된 게임 세이브 데이터 및 설정은 기기 내 앱 데이터 삭제 또는 앱 제거를 통해 직접 삭제할 수 있습니다 (Android: 설정 → 앱 → 퍼즐 킹덤 → 저장공간 → 데이터 삭제).

2. 이메일로 삭제 요청하는 방법

  1. 아래 이메일 주소로 삭제 요청 메일을 보냅니다.
  2. 이메일 제목: [퍼즐 킹덤] 계정 및 데이터 삭제 요청
  3. 본문에 포함할 정보 (계정 확인에 필요한 최소한의 정보만 요청드립니다):
    • 게임 내 닉네임 또는 계정 식별에 도움이 되는 정보 (있는 경우)
    • 요청 내용: 전체 데이터 삭제 또는 특정 데이터 삭제
  4. 포함하지 말 것: 비밀번호, 결제 카드 정보, 주민등록번호 등 인증 자격 증명 및 민감 금융 정보. NextFlow는 이런 정보를 요청하지 않습니다.

3. 삭제 처리 범위

  • 서버에 저장된 게임 진행 데이터 (활성화된 경우)
  • 구매 처리 및 검증 기록 (보존 의무 제외 범위)
  • 문의 이메일 기록
  • 계정 식별 정보 (계정 기능 활성화된 경우)

4. 즉시 삭제되지 않을 수 있는 데이터

다음의 경우 해당 정보가 일정 기간 별도 보관될 수 있습니다:

  • 법령상 보존 의무가 있는 결제·거래 기록
  • 보안 사고 대응 또는 부정 이용 방지를 위해 필요한 최소 정보
  • 진행 중인 결제 분쟁 또는 환불 처리에 필요한 정보

위의 경우 해당 보존 기간 만료 후 파기됩니다. 요청 처리 결과 및 보존 예외 사유가 있는 경우 이메일로 안내해 드립니다.

5. 처리 기간

요청은 접수 후 합리적으로 가능한 시일 내에 검토·처리합니다. 처리가 완료되면 요청하신 이메일로 결과를 안내해 드립니다.

(법적으로 보장된 처리 기한은 현재 이 문서에 명시되어 있지 않습니다. 운영 주체가 확정된 서비스 약관 또는 관련 법령 기한을 확인 후 이 페이지를 업데이트해야 합니다.)
