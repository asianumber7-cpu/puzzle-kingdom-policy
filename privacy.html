1. 개요

퍼즐 킹덤(com.nextflow.puzzlekingdom)은 NextFlow가 운영하는 퍼즐 RPG 모바일 게임입니다. 본 개인정보처리방침은 게임 이용 과정에서 수집·이용·보관·파기되는 정보에 대해 설명합니다.

2. 처리하는 정보

  • 앱 내 구매 및 영수증 데이터: 결제 처리 및 부정 이용 방지 목적으로 Google Play 영수증 정보를 처리합니다. 서버 측 결제 검증(Cloud Functions IAP 검증 seam) 기능이 준비되어 있으며, 활성화 시 검증 목적으로 영수증 요약 정보가 서버로 전송될 수 있습니다. 현재 활성화 여부는 오너가 최종 확인 후 이 방침을 업데이트해야 합니다.
  • 로컬 게임 진행 데이터 및 설정: 게임 세이브 데이터(레벨 진행도, 보유 자원, 영웅 보유 현황, 설정 값 등)는 기기 내부 저장소에 로컬로 보관됩니다. 클라우드 동기화 여부는 오너가 확인 후 이 방침에 추가해야 합니다.
  • 문의 이메일 내용: 사용자가 asianumber7@gmail.com으로 문의·지원 요청을 보내는 경우, 해당 이메일 내용(문의 내용, 발신 이메일 주소)이 처리됩니다.
  • 계정·데이터 삭제 요청 정보: 사용자가 삭제를 요청하는 경우, 요청 확인에 필요한 최소한의 정보(예: 게임 내 식별 정보)를 처리합니다.
  • 분석 데이터: 현재 레포 코드에 Firebase Analytics 어댑터(FirebaseAnalyticsAdapter.cs)가 존재하나 이벤트 전송 본문이 전부 비활성화(주석 처리)되어 있어 실제 수집이 이루어지지 않습니다. 분석 기능 활성화 시 본 방침과 Play 데이터 보안 선고를 업데이트해야 합니다.
  • 광고 데이터: 현재 광고 서비스는 시뮬레이션 구현(FakeAdService)으로 실제 광고 SDK가 연결되어 있지 않습니다. 광고 SDK 연결 시 본 방침과 Play 데이터 보안 선고를 업데이트해야 합니다.

4. 개인정보 제3자 제공 및 판매

NextFlow는 사용자의 개인정보를 제3자에게 판매하지 않습니다. 결제 처리 목적에 한해 Google Play(Google LLC)와의 영수증 검증 과정에서 최소한의 영수증 정보가 전달될 수 있습니다. 법령에 의해 요구되는 경우 또는 사용자의 명시적 동의가 있는 경우를 제외하고 다른 목적으로 개인정보를 공유하지 않습니다.

5. 보안

서버와의 통신(Cloud Functions 등)은 HTTPS를 통해 암호화됩니다. 기기 내 로컬 저장 데이터의 암호화 여부는 오너가 확인 후 이 방침에 추가해야 합니다. 부정 이용 방지를 위한 합리적인 기술적 조치를 적용합니다.

6. 보관 및 파기

  • 로컬 게임 세이브 데이터는 앱 데이터 삭제 또는 앱 제거 시 기기에서 삭제됩니다.
  • 서버에 보관된 데이터(IAP 영수증 처리 기록, 문의 이메일 등)는 처리 목적 달성 후 법령 보존 의무 범위 내에서 최소 기간 보관 후 파기합니다.
  • 결제 분쟁 대응, 부정 이용 방지, 법령상 보존 의무가 있는 정보는 해당 기간 동안 별도 보관 후 파기합니다.
